# 🤖 AI-Assisted Smart Recruitment System

An AI-powered recruitment system that automates resume screening, skill extraction, and candidate-job matching using Machine Learning, NLP, and Gradio UI. The project improves hiring efficiency through intelligent resume analysis and job role prediction.

## 🚀 Technologies Used

🐍 Python • 🧠 Machine Learning • 📄 NLP • 📊 Scikit-learn • 🎨 Gradio
